GitExo est un site non-officiel a propos de Git , mis en place dans le cadre d’un projet en binome pour Molengeek. Le but est simple: l’apprentissage par l’action.Nous avons donc rassemblés quelques exercices avec leurs explicatifs que vous pouvez télécharger pour vous exercer. Que vous soyez débutant, ou que vous cherchez à vous perfectionner, GitExo est une solution simple, efficace et entièrement gratuite!

A PROPOS

Git est un système de contrôle de version, gratuit et open source utilisé dans le monde entier. Reconnu par les professionnels de plusieurs secteurs pour son coté pratique et efficace, Git est aujourd’hui devenu incontournable pour quiconque s’intérèsse aux code ou à la programation. C’est pourquoi GitExo a centraliser ici même divers exercices pour vous permettre de progresser a votre rythme.
