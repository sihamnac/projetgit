A PROPOS DE GIT

Git est un système de contrôle de version, gratuit et open source utilisé dans le monde entier. Reconnu par les professionnels de plusieurs secteurs pour son coté pratique et efficace, Git est aujourd’hui devenu incontournable pour quiconque s’intérèsse aux code ou à la programation. C’est pourquoi GitExo a centraliser ici même divers exercices pour vous permettre de progresser a votre rythme.

LE PRINCIPE DE GIT

Le principe de Git est assez simple. Imaginez un projet sous la forme d'un arbre. Chacun des collaborateurs est affecté a une branche sur laquelle il developpe son travail. A chaque étape de son travail, il peut effectuer une sauvegarde et celle-ci au lieu d'écraser les précédente sauvegarde va s'ajouter pour former une série de noueds, les "Commits" sur lesquelles il est possible de revenir pour récupérer le travail a l'état du moment de la sauvegarde. lorsque vous poussez vers un référentiel distant, vous n’avez pas besoin de pousser toutes vos branches. Vous pouvez choisir pour partager une seule de vos branches, quelques-unes d’entre elles ou toutes. Cela tend à libérer les gens pour qu’ils essaient de nouveaux sans se soucier d’avoir à planifier comment et quand ils vont les fusionner ou les partager avec autrui.
